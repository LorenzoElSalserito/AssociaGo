# 7. Inventarverwaltung (Inventory)

Der Bereich **Inventar** dient dazu, die materiellen Vermögenswerte des Vereins zu verfolgen.

## 7.1 Gegenstandsliste
Listet alle Vermögenswerte auf, mit Menge, Wert und Status.

## 7.2 Gegenstand hinzufügen
Das Formular ermöglicht die Eingabe von:
- **Name und Inventarcode**.
- **Kategorie**: (Elektronik, Möbel, Sport, etc.).
- **Menge und Wert**.
- **Zustand**: (Neu, Gut, Defekt, Entsorgt).
- **Erwerbsmethode**: (Kauf, Spende, Leihgabe).
    - *Hinweis*: Wenn "Kauf" ausgewählt ist, schlägt das System vor, automatisch die Ausgabentransaktion zu erstellen.

## 7.3 Ausleihen (Loans)
Von der Karte eines Gegenstands aus können Sie Ausleihen an Mitglieder verwalten:
- **Ausleihe registrieren**: Wählen Sie das Mitglied und das Datum aus.
- **Rückgabe**: Markieren Sie den Gegenstand als zurückgegeben und aktualisieren Sie die Verfügbarkeit.
- **Verlauf**: Sehen Sie alle Bewegungen des Gegenstands an.
