# 9. Gutscheine und Rabatte (Coupons)

Der Bereich **Gutscheine** ermöglicht das Erstellen von Rabattcodes für Aktivitäten und Veranstaltungen.

## 9.1 Gutschein erstellen
- **Code**: Der einzugebende Text (z.B. "SOMMER2024").
- **Rabattart**: Prozentual oder Festbetrag.
- **Wert**: Die Höhe des Rabatts.
- **Gültigkeit**: Start- und Enddatum.
- **Limits**: Maximale Anzahl der Nutzungen.
- **Anwendbarkeit**: Kann auf bestimmte Aktivitäten oder Veranstaltungen beschränkt werden.

## 9.2 Nutzung
Gutscheine können während der Teilnehmerregistrierung für eine Veranstaltung oder Aktivität angewendet werden, wodurch der fällige Betrag automatisch reduziert wird.
