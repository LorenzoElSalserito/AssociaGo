# 10. Einstellungen (Settings)

Das Bedienfeld zur Konfiguration der gesamten Plattform.

## 10.1 Vereinsprofil
- **Identitätsdaten**: Name, Steuernummer, Umsatzsteuer-ID.
- **Kontakte**: Adresse, E-Mail, Telefon.
- **Logo**: Hochladen des Bildes, das auf allen Dokumenten erscheinen wird.
- **Organigramm**: Namen der sozialen Ämter (Präsident, Schatzmeister, etc.) für Protokolle.

## 10.2 Konfiguration
- **Kartenformat**: Anpassung des Codes (z.B. "2024-####").
- **Grundgebühr**: Standardbetrag für die Anmeldung.

## 10.3 Datenbank
- **Lokal (SQLite)**: Standard, Datei auf dem Gerät gespeichert.
- **Remote (MariaDB)**: Konfiguration für die Verbindung zu einem externen Server (Host, Port, Benutzer, Passwort).

## 10.4 Backup
- **Lokales Backup**: Erstellt eine sofortige Sicherheitskopie der SQLite-Datenbank.

## 10.5 Erscheinungsbild
- **Sprache**: Italienisch, Englisch, Deutsch, Spanisch, Französisch.
- **Thema**: Hell oder Dunkel.
