# 16. Amtliche Unterschriften

Der Bereich **Unterschriften** (erreichbar über Einstellungen > Unterschriften) ermöglicht die Konfiguration der visuellen Unterschriften der amtlichen Vertreter des Vereins.

## 16.1 Wozu dienen sie
Die amtlichen Unterschriften werden automatisch angebracht auf:
- **Jahresabschlüsse** (PDF)
- **Bescheinigungen und Zertifikate** (PDF)
- **Versammlungsprotokolle** (wenn erstellt)
- Alle offiziellen Dokumente, die von der Plattform erzeugt werden.

## 16.2 Verfügbare Rollen
AssociaGo verwaltet drei Unterzeichnerrollen:
- **Vorsitzender**: Die Hauptunterschrift der Organisation.
- **Schriftführer**: Die Unterschrift des Protokollführers.
- **Schatzmeister**: Die Unterschrift des Finanzverantwortlichen.

## 16.3 Konfiguration

### Für jede Rolle
In der Registerkarte Unterschriften der Einstellungen finden Sie für jede Rolle eine Karte mit:

1. **Name des Unterzeichners**: Der vollständige Name der Person, die das Amt bekleidet (z. B. "Max Mustermann").
2. **Titel/Funktion**: Die offizielle Bezeichnung (z. B. "Vorsitzender des Vorstands").
3. **Unterschriftsbild**: Upload einer PNG- oder JPG-Datei mit dem Scan der handschriftlichen Unterschrift.

### Eine Unterschrift hochladen
1. Klicken Sie auf **"Bild hochladen"** in der Karte der gewünschten Rolle.
2. Wählen Sie eine Bilddatei (PNG, JPG) mit der Unterschrift aus.
3. Das Bild wird hochgeladen und als Vorschau angezeigt.
4. Klicken Sie auf **"Speichern"** zur Bestätigung.

### Eine Unterschrift entfernen
Klicken Sie auf das **Papierkorb**-Symbol neben der Vorschau, um das Unterschriftsbild zu entfernen.

## 16.4 Status der Unterschriften
Unterschriften haben zwei Status:
- **Aktiv**: Die Unterschrift ist für die Anbringung auf Dokumenten verfügbar.
- **Inaktiv**: Die Unterschrift ist archiviert und wird nicht verwendet.

## 16.5 Empfehlungen
- Verwenden Sie Bilder mit **transparentem Hintergrund** (PNG) für ein besseres Ergebnis auf Dokumenten.
- Die empfohlene Auflösung beträgt mindestens **300x100 Pixel**.
- Halten Sie Namen und Titel bei Neubesetzung der Vereinsämter aktuell.
- Konfigurieren Sie die Unterschriften **vor** der Erstellung von Abschlüssen, Bescheinigungen oder offiziellen Protokollen.

> **Wichtig**: Die in diesem Bereich konfigurierten Unterschriften sind visuelle Unterschriften (Bilder). Sie haben nicht den Wert einer qualifizierten elektronischen Signatur im Sinne der eIDAS-Verordnung. Für Dokumente, die eine digitale Signatur erfordern, verwenden Sie zertifizierte elektronische Signaturwerkzeuge.
