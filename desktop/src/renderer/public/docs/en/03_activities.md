# 3. Activity Management (Activities)

The **Activities** section is designed to manage courses, workshops, seminars, and any other ongoing or long-term initiative.

## 3.1 Activity List
The main table shows:
- **Activity Name**
- **Schedule**: A summary of weekly times (e.g., "Mon/Wed 18-20").
- **Participants**: Number of enrolled members.
- **Status**: (Active, Archived).

## 3.2 Creating a New Activity
The creation form is comprehensive and allows you to define every aspect:

### Basic Information
- **Name and Description**: To identify the activity.
- **Category**: (Course, Workshop, Seminar, Other).
- **Subscription Type**:
    - **One Time**: Single payment.
    - **Monthly**: Recurring payment.
    - **Annual**: Annual payment.
- **Cost**: The base amount for enrollment.

### Operational Details
- **Dates**: Start and end (optional).
- **Schedule**: A text field to describe the times.
- **Location**: Where the activity takes place.
- **Max Participants**: Maximum limit of enrollees.

### Advanced Options
- **Require Registration**: If unchecked, the activity is informational only.
- **Generate Invoice**: If selected, the system can generate fiscal documents.
- **Image/Document Paths**: To attach visual or informational material.

## 3.3 Activity Detail
Clicking on an activity takes you to a dedicated dashboard with 4 tabs:

### 1. Participants
- **List**: List of all enrolled members.
- **Add**: Allows enrolling a member in the activity.
- **Payment Status**: Indicates if the participant has paid the fee.

### 2. Schedule
- **List**: List of weekly times.
- **Add**: Allows defining a new time slot with:
    - Day of the week.
    - Start and end time.
    - Location (if different from the main one).

### 3. Instructors
- **List**: List of associated instructors.
- **Add**: Allows adding an instructor with:
    - Personal data.
    - Specialization.
    - **Compensation Type**: (Hourly, Flat Rate, Volunteer).
    - **Compensation Value**: The hourly or flat rate amount.

### 4. Costs
- **List**: List of costs incurred for the activity (rent, materials, etc.).
- **Add**: Allows registering a new cost with:
    - Description and Amount.
    - Category (Equipment, Venue, Marketing, etc.).
    - **Recurring Cost**: If the cost repeats (e.g., monthly rent).

## 3.4 Activity Report
From the detail page, you can download a complete **PDF report** summarizing:
- Activity data.
- Participant list.
- Financial summary (income vs costs).
- Schedule and instructors.
