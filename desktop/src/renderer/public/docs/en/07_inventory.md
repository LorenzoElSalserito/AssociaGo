# 7. Inventory Management (Inventory)

The **Inventory** section is used to track the association's material assets.

## 7.1 Item List
Lists all assets, with quantity, value, and status.

## 7.2 Add Item
The form allows you to enter:
- **Name and Inventory Code**.
- **Category**: (Electronics, Furniture, Sports, etc.).
- **Quantity and Value**.
- **Condition**: (New, Good, Broken, Disposed).
- **Acquisition Method**: (Purchase, Donation, Loan).
    - *Note*: If "Purchase" is selected, the system proposes to automatically create the expense transaction.

## 7.3 Loans
From an item's card, you can manage loans to members:
- **Register Loan**: Select the member and the date.
- **Return**: Mark the item as returned and update availability.
- **History**: View all movements of the item.
