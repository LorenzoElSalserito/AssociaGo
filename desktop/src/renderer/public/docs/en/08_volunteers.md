# 8. Volunteer Management (Volunteers)

The **Volunteers** section allows organizing the association's workforce.

## 8.1 Volunteer Registry
Links a member to the volunteer role, specifying:
- **Skills**: (e.g., First Aid, Cooking, Driver).
- **Availability**: (e.g., Weekends, Evenings).

## 8.2 Shifts
In the volunteer's card, you can assign work shifts:
- **Date and Time**.
- **Specific Role**.
- **Event/Activity**: Optional link to a specific event.
- **Status**: (Scheduled, Completed, Missed).

## 8.3 Expense Reimbursements
Management of reimbursement requests for volunteers:
- Expense registration (with description and amount).
- Approval/Rejection.
- Payment (creates outgoing transaction).
