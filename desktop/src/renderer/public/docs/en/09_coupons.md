# 9. Coupons and Discounts (Coupons)

The **Coupons** section allows creating discount codes for activities and events.

## 9.1 Creating a Coupon
- **Code**: The text to enter (e.g., "SUMMER2024").
- **Discount Type**: Percentage or Fixed.
- **Value**: The amount of the discount.
- **Validity**: Start and end dates.
- **Limits**: Maximum number of uses.
- **Applicability**: Can be limited to specific activities or events.

## 9.2 Usage
Coupons can be applied during participant registration for an event or activity, automatically reducing the amount due.
