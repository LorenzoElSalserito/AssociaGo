# 10. Settings

The control panel to configure the entire platform.

## 10.1 Association Profile
- **Identity Data**: Name, Tax Code, VAT Number.
- **Contacts**: Address, Email, Phone.
- **Logo**: Upload the image that will appear on all documents.
- **Organization Chart**: Names of social offices (President, Treasurer, etc.) for minutes.

## 10.2 Configuration
- **Card Format**: Customization of the code (e.g., "2024-####").
- **Base Fee**: Standard amount for enrollment.

## 10.3 Database
- **Local (SQLite)**: Default, file saved on the device.
- **Remote (MariaDB)**: Configuration for connection to external server (Host, Port, User, Password).

## 10.4 Backup
- **Local Backup**: Creates an immediate backup copy of the SQLite database.

## 10.5 Appearance
- **Language**: Italian, English, German, Spanish, French.
- **Theme**: Light or Dark.
