# 9. Cupones y Descuentos (Coupons)

La sección **Cupones** permite crear códigos de descuento para actividades y eventos.

## 9.1 Creación de Cupón
- **Código**: El texto a introducir (ej. "VERANO2024").
- **Tipo de Descuento**: Porcentual o Fijo.
- **Valor**: La entidad del descuento.
- **Validez**: Fechas de inicio y fin.
- **Límites**: Número máximo de usos.
- **Aplicabilidad**: Puede limitarse a actividades o eventos específicos.

## 9.2 Uso
Los cupones se pueden aplicar durante la inscripción de un participante a un evento o actividad, reduciendo automáticamente el importe debido.
