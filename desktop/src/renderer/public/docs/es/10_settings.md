# 10. Ajustes (Settings)

El panel de control para configurar toda la plataforma.

## 10.1 Perfil de la Asociación
- **Datos Identificativos**: Nombre, Código Fiscal, Número de IVA.
- **Contactos**: Dirección, Correo, Teléfono.
- **Logotipo**: Carga de la imagen que aparecerá en todos los documentos.
- **Organigrama**: Nombres de los cargos sociales (Presidente, Tesorero, etc.) para las actas.

## 10.2 Configuración
- **Formato de Tarjeta**: Personalización del código (ej. "2024-####").
- **Cuota Base**: Importe estándar para la inscripción.

## 10.3 Base de Datos
- **Local (SQLite)**: Predeterminado, archivo guardado en el dispositivo.
- **Remota (MariaDB)**: Configuración para conexión a servidor externo (Host, Puerto, Usuario, Contraseña).

## 10.4 Copia de Seguridad
- **Copia de Seguridad Local**: Crea una copia de seguridad inmediata de la base de datos SQLite.

## 10.5 Apariencia
- **Idioma**: Italiano, Inglés, Alemán, Español, Francés.
- **Tema**: Claro u Oscuro.
