# Manuel de l'Utilisateur AssociaGo

Bienvenue dans le manuel officiel d'**AssociaGo**, la plateforme de gestion complète pour les associations (APS, ASD, ODV, ETS).
Ce guide a été rédigé pour accompagner l'utilisateur étape par étape dans l'utilisation de chaque fonctionnalité du logiciel.

## 1. Le Tableau de Bord Principal

Le **Tableau de Bord** est le point d'atterrissage après la connexion. Il est conçu pour offrir une vue d'ensemble immédiate de l'état de santé de l'association.

### 1.1 En-tête (Header)
En haut de la page, vous trouverez :
- **Nom de l'Association** : Le nom configuré dans les paramètres.
- **Logo** : Le logo officiel de l'association (si téléchargé).
- **Date Actuelle** : La date complète d'aujourd'hui.

### 1.2 Indicateurs Clés de Performance (KPI)
Sous l'en-tête, trois cartes principales affichent les données vitales :

1.  **Membres Actifs (Active Members)** :
    *   Affiche le nombre total de membres ayant une adhésion active (non expirée) et à jour de leurs paiements.
    *   Un indicateur vert montre combien de nouveaux membres se sont inscrits au cours du mois en cours.
    *   *Note* : Cette donnée est fondamentale pour comprendre la tendance de croissance de l'association.

2.  **Solde de Trésorerie (Treasury Balance)** :
    *   Affiche le solde actuel calculé en additionnant tous les revenus et en soustrayant toutes les dépenses enregistrées dans le système.
    *   Sous le solde, les mouvements du mois en cours sont affichés :
        *   **Vert** : Total des revenus du mois.
        *   **Rouge** : Total des dépenses du mois.

3.  **Événements à Venir (Upcoming Events)** :
    *   Indique le nombre d'événements prévus pour le futur.
    *   Affiche le détail immédiat du **prochain événement** dans le calendrier (Titre et Date), permettant de ne pas manquer les échéances importantes.

### 1.3 Activité Récente
La section centrale liste chronologiquement les dernières opérations effectuées sur la plateforme, telles que :
- Enregistrement de nouveaux paiements.
- Inscription de nouveaux membres.
- Création de nouveaux événements.
Ce journal aide les administrateurs à suivre les opérations quotidiennes.

### 1.4 Actions Rapides (Quick Actions)
À droite, un panneau bleu offre des raccourcis pour les opérations les plus fréquentes, évitant d'avoir à naviguer dans les menus :
- **Inscrire un Nouveau Membre** : Ouvre directement le formulaire d'inscription.
- **Créer un Événement** : Ouvre le formulaire pour créer un nouvel événement.
- **Enregistrer un Paiement** : Vous emmène directement à la section Finance pour saisir une transaction.
