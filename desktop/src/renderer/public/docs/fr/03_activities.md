# 3. Gestion des Activités (Activities)

La section **Activités** est conçue pour gérer les cours, ateliers, séminaires et toute autre initiative continue ou à long terme.

## 3.1 Liste des Activités
Le tableau principal affiche :
- **Nom de l'Activité**
- **Calendrier** : Un résumé des horaires hebdomadaires (ex. "Lun/Mer 18-20").
- **Participants** : Nombre de membres inscrits.
- **Statut** : (Active, Archivée).

## 3.2 Créer une Nouvelle Activité
Le formulaire de création est complet et vous permet de définir chaque aspect :

### Informations de Base
- **Nom et Description** : Pour identifier l'activité.
- **Catégorie** : (Cours, Atelier, Séminaire, Autre).
- **Type d'Abonnement** :
    - **Unique** : Paiement unique.
    - **Mensuel** : Paiement récurrent.
    - **Annuel** : Paiement annuel.
- **Coût** : Le montant de base pour l'inscription.

### Détails Opérationnels
- **Dates** : Début et fin (optionnel).
- **Calendrier** : Un champ texte pour décrire les horaires.
- **Lieu** : Où se déroule l'activité.
- **Max Participants** : Limite maximale d'inscrits.

### Options Avancées
- **Inscription Requise** : Si décoché, l'activité est uniquement informative.
- **Générer Facture** : Si sélectionné, le système peut générer des documents fiscaux.
- **Chemins Image/Document** : Pour joindre du matériel visuel ou informatif.

## 3.3 Détail de l'Activité
En cliquant sur une activité, vous accédez à un tableau de bord dédié avec 4 onglets :

### 1. Participants
- **Liste** : Liste de tous les membres inscrits.
- **Ajouter** : Permet d'inscrire un membre à l'activité.
- **Statut de Paiement** : Indique si le participant a payé les frais.

### 2. Calendrier (Schedule)
- **Liste** : Liste des horaires hebdomadaires.
- **Ajouter** : Permet de définir un nouveau créneau horaire avec :
    - Jour de la semaine.
    - Heure de début et de fin.
    - Lieu (si différent du principal).

### 3. Instructeurs
- **Liste** : Liste des instructeurs associés.
- **Ajouter** : Permet d'ajouter un instructeur avec :
    - Données personnelles.
    - Spécialisation.
    - **Type de Rémunération** : (Horaire, Forfaitaire, Bénévole).
    - **Valeur de Rémunération** : Le montant horaire ou forfaitaire.

### 4. Coûts
- **Liste** : Liste des coûts engagés pour l'activité (loyer, matériel, etc.).
- **Ajouter** : Permet d'enregistrer un nouveau coût avec :
    - Description et Montant.
    - Catégorie (Équipement, Lieu, Marketing, etc.).
    - **Coût Récurrent** : Si le coût se répète (ex. loyer mensuel).

## 3.4 Rapport d'Activité
Depuis la page de détail, vous pouvez télécharger un **rapport PDF** complet résumant :
- Données de l'activité.
- Liste des participants.
- Résumé financier (revenus vs coûts).
- Calendrier et instructeurs.
