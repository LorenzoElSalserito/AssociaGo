# 8. Gestion des Bénévoles (Volunteers)

La section **Bénévoles** permet d'organiser la force de travail de l'association.

## 8.1 Registre des Bénévoles
Lie un membre au rôle de bénévole, en spécifiant :
- **Compétences** : (ex. Premiers Secours, Cuisine, Chauffeur).
- **Disponibilité** : (ex. Week-ends, Soirées).

## 8.2 Quarts (Shifts)
Dans la fiche du bénévole, vous pouvez attribuer des quarts de travail :
- **Date et Heure**.
- **Rôle spécifique**.
- **Événement/Activité** : Lien optionnel vers un événement spécifique.
- **Statut** : (Prévu, Terminé, Manqué).

## 8.3 Remboursements de Frais
Gestion des demandes de remboursement pour les bénévoles :
- Enregistrement de la dépense (avec description et montant).
- Approbation/Rejet.
- Paiement (crée une transaction sortante).
