# 10. Paramètres (Settings)

Le panneau de contrôle pour configurer l'ensemble de la plateforme.

## 10.1 Profil de l'Association
- **Données d'Identité** : Nom, Code Fiscal, Numéro de TVA.
- **Contacts** : Adresse, Email, Téléphone.
- **Logo** : Téléchargement de l'image qui apparaîtra sur tous les documents.
- **Organigramme** : Noms des charges sociales (Président, Trésorier, etc.) pour les procès-verbaux.

## 10.2 Configuration
- **Format de Carte** : Personnalisation du code (ex. "2024-####").
- **Frais de Base** : Montant standard pour l'inscription.

## 10.3 Base de Données
- **Locale (SQLite)** : Par défaut, fichier enregistré sur l'appareil.
- **Distante (MariaDB)** : Configuration pour la connexion à un serveur externe (Hôte, Port, Utilisateur, Mot de passe).

## 10.4 Sauvegarde
- **Sauvegarde Locale** : Crée une copie de sécurité immédiate de la base de données SQLite.

## 10.5 Apparence
- **Langue** : Italien, Anglais, Allemand, Espagnol, Français.
- **Thème** : Clair ou Sombre.
