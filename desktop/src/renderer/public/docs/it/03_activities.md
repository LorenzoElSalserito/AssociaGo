# 3. Gestione Attività (Activities)

La sezione **Attività** è pensata per gestire corsi, workshop, seminari e qualsiasi altra iniziativa continuativa o a lungo termine.

## 3.1 Lista Attività
La tabella principale mostra:
- **Nome Attività**
- **Orari**: Un riepilogo degli orari settimanali (es. "Lun/Mer 18-20").
- **Partecipanti**: Numero di iscritti.
- **Stato**: (Attiva, Archiviata).

## 3.2 Creare una Nuova Attività
Il modulo di creazione è completo e permette di definire ogni aspetto:

### Informazioni di Base
- **Nome e Descrizione**: Per identificare l'attività.
- **Categoria**: (Corso, Workshop, Seminario, Altro).
- **Tipo di Sottoscrizione**:
    - **Una Tantum**: Pagamento unico.
    - **Mensile**: Pagamento ricorrente.
    - **Annuale**: Pagamento annuale.
- **Costo**: L'importo base per l'iscrizione.

### Dettagli Operativi
- **Date**: Inizio e fine (opzionale).
- **Orari**: Un campo testuale per descrivere gli orari.
- **Luogo**: Dove si svolge l'attività.
- **Max Partecipanti**: Limite massimo di iscritti.

### Opzioni Avanzate
- **Richiede Registrazione**: Se deselezionato, l'attività è solo informativa.
- **Genera Fattura**: Se selezionato, il sistema può generare documenti fiscali.
- **Percorsi Immagine/Documento**: Per allegare materiale visivo o informativo.

## 3.3 Dettaglio Attività
Cliccando su un'attività si accede a una dashboard dedicata con 4 tab:

### 1. Partecipanti
- **Elenco**: Lista di tutti i soci iscritti.
- **Aggiungi**: Permette di iscrivere un socio all'attività.
- **Stato Pagamento**: Indica se il partecipante ha saldato la quota.

### 2. Orari (Schedule)
- **Elenco**: Lista degli orari settimanali.
- **Aggiungi**: Permette di definire un nuovo slot orario con:
    - Giorno della settimana.
    - Ora di inizio e fine.
    - Luogo (se diverso da quello principale).

### 3. Istruttori
- **Elenco**: Lista degli istruttori associati.
- **Aggiungi**: Permette di aggiungere un istruttore con:
    - Dati anagrafici.
    - Specializzazione.
    - **Tipo di Compenso**: (Orario, Forfettario, Volontario).
    - **Valore Compenso**: L'importo orario o forfettario.

### 4. Costi
- **Elenco**: Lista dei costi sostenuti per l'attività (affitto, materiali, etc.).
- **Aggiungi**: Permette di registrare un nuovo costo con:
    - Descrizione e Importo.
    - Categoria (Attrezzatura, Sede, Marketing, etc.).
    - **Costo Ricorrente**: Se il costo si ripete (es. affitto mensile).

## 3.4 Report Attività
Dalla pagina di dettaglio, è possibile scaricare un **report PDF** completo che riassume:
- Dati dell'attività.
- Lista partecipanti.
- Riepilogo finanziario (entrate vs costi).
- Orari e istruttori.
