# 10. Impostazioni (Settings)

Il pannello di controllo per configurare l'intera piattaforma.

## 10.1 Profilo Associazione
- **Dati Identificativi**: Nome, CF, P.IVA.
- **Contatti**: Indirizzo, Email, Telefono.
- **Logo**: Caricamento dell'immagine che apparirà su tutti i documenti.
- **Organigramma**: Nomi delle cariche sociali (Presidente, Tesoriere, etc.) per i verbali.

## 10.2 Configurazione
- **Formato Tessera**: Personalizzazione del codice (es. "2024-####").
- **Quota Base**: Importo standard per l'iscrizione.

## 10.3 Database
- **Locale (SQLite)**: Default, file salvato sul dispositivo.
- **Remoto (MariaDB)**: Configurazione per connessione a server esterno (Host, Port, User, Password).

## 10.4 Backup
- **Backup Locale**: Crea una copia di sicurezza immediata del database SQLite.

## 10.5 Aspetto
- **Lingua**: Italiano, Inglese, Tedesco, Spagnolo, Francese.
- **Tema**: Chiaro o Scuro.
